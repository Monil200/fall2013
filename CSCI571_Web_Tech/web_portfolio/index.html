Projects

Project 1--Yahoo! Stock Search--(Slides 1-3)

The Yahoo! Stock Search application fetches real time stock quotes, stock chart and news headlines of any company using Yahoo's APIs. It uses YUI, Amazon's AWS ,JDOM for XML parsing and Facebook share capability to share the company quotes on users wall. The PHP script hosted on AWS returns XML which is parsed by Servlets and servlets returns JSON to the front end call made by Javascript.

Project 2--JobQuest - A Mashup--(Slides 4-5)

JobQuest is a job aggregator portal which integrates jobs from portals like Glassdoor, Monster and Jobdiagnosis. These protal are crawled on regular basis to update the job data store which is in the form of JSON files. The resulting mashup (gathering data from more than one sources to create a new application) is more detailed than any of the current job portal in the world.

Project 3--Hybrid AES/DES Encryption/Decryption Algorithm--(Slides 6-10)

The hybrid AES/DES is anovel approach to information ecnrption which can be in form of strings, images,videos and audios. The key strength achieved using this approach is more than the two algorithm's actual strength. To add to the key strength a novel theory of irrational numbers is used which makes the algorithm uncrackable.

Project 4--Extreme Downloader and Manager--(Slides 4-5)

The Downlaod manager is a tool that accelerates the download by downloading multiple segements of the file together. It send requests for differents parts of the same file to downlaod different parts simultaneoulsy. It has options to pause, stop, resume and schedule downlaod at a later time. The downlaod window shows the status of the current download along with the status of the individual segments.
